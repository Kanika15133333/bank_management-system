# Bank Management System

A C++ project developed using Object-Oriented Programming and File Handling.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Search Account
- Delete Account
- Persistent Storage using Files

## Technologies Used
- C++
- OOP Concepts
- File Handling

## How to Run

g++ main.cpp BankAccount.cpp -o bank

./bank
